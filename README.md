# 📊 Adventure Works Sales Analysis (2024–2026)

## 🔍 Project Overview
This project analyzes the Adventure Works dataset to evaluate business performance, customer purchasing behavior, and product-level revenue drivers. The goal is to identify high-value segments and support data-driven decision-making.

## 🎯 Objectives
- Identify key revenue drivers across product models and colors  
- Analyze customer demographics (Marital Status, Education, Occupation)  
- Track KPIs such as total sales, orders, and growth rate
- 
## 📁 Data Sources
- Dataset: Adventure Works Retail Database  
- Timeline: 2024–2026  
- Tables: Sales, Customers, Products, Territories  
- Size: 18,000+ rows  

## 🧹 Data Cleaning (Power Query)
- Standardized categorical values (Gender, Marital Status, Home Ownership)  
- Handled missing values (Product Color → "Black")  
- Corrected data types (Currency, Text, Date)  
- Applied Trim & Clean transformations  

## 🔄 Data Transformation
- Created calculated columns (Age, Monthly Salary)  
- Built Calendar Table using DAX (Year, Month, Quarter)  
- Designed Star Schema Data Model  

## 📊 Dashboard Features
- Slicers: Occupation 
- Visuals:
  - Bar Chart → Product Models  
  - Pie Chart → Education  
  - Funnel Chart → Product Colors  
  - Scatter Plot → Customer behavior  

## 📈 Key Metrics
- Total Sales: **$39M**  
- Total Orders: **18K**  
- Growth Rate: **11%**  
- Max Sale Value: **$17.89K**
- 
## 💡 Key Insights
- Road-150 is the top-performing product (Avg. Sale: $10.9K)  
- Married customers contribute ~**$21M** in revenue  
- Professionals are the highest-value segment  
- Black and Red products dominate sales
- 
## 🧠 Analytics Approach
- Descriptive → Performance overview  
- Diagnostic → Revenue driven by Professionals & Married customers  
- Predictive → Forecast ~$43M sales by 2027  
- Prescriptive → Focus marketing on high-income segments  

## ✅ Conclusion
High-end bike models and specific customer segments (Professional, Married) are the primary revenue drivers. 
The insights support targeted marketing strategies and optimized inventory planning to sustain growth.
